## 1. Ce que fait Navve

Navve vous aide à enregistrer, organiser et partager les adresses qui comptent pour vous. Ajoutez-les à votre façon : partagez directement depuis Apple Maps, collez un lien Apple Maps, saisissez une adresse ou remplissez les détails manuellement. L'app fonctionne sur votre appareil et, si vous êtes connecté à iCloud, vos données sont automatiquement sauvegardées (vous pouvez désactiver cette option dans les réglages de votre appareil si vous préférez).

Les **collections** vous permettent de regrouper vos adresses enregistrées dans des listes nommées — par exemple, « Coins du week-end » ou « Restaurants à tester ». Vous pouvez ajouter une adresse à une ou plusieurs collections, les réordonner et partager une collection avec d'autres.

Le **mode itinéraire** est un mode spécial conçu pour les déplacements. Lorsque vous l'activez pour une collection, vous pouvez attribuer chaque adresse à un jour précis et à un créneau horaire (Matin, Déjeuner, Après-midi, Dîner, Soirée), définir vos dates de voyage, et exporter un itinéraire jour par jour partageable.

## 2. Formules (Gratuite & Pro)

Navve propose deux formules :

- **Formule Gratuite** : enregistrez jusqu'à 8 adresses avec sauvegarde iCloud entre vos appareils. Toutes les fonctionnalités sont débloquées ; seul le nombre d'adresses enregistrées est limité.
- **Formule Pro** : achat unique — adresses illimitées avec synchronisation iCloud sur tous vos appareils. Pas d'abonnement ; payez une fois et gardez Pro pour toujours.
- **Les achats sont gérés via l'App Store d'Apple**. Pro est un achat intégré non consommable (identifiant de produit : OneTimePayment).
- **Restaurer l'achat** : si vous réinstallez l'app ou changez d'appareil, utilisez « Restaurer l'achat » dans les Réglages ou depuis la fenêtre de mise à niveau pour retrouver votre statut Pro.
- **Utiliser un code** : si vous avez une carte cadeau ou un code promotionnel, utilisez « Utiliser un code » dans Réglages → Abonnement ou depuis la fenêtre de mise à niveau pour activer Pro.
