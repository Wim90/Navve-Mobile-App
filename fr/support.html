FAQ

Questions fréquentes

Comment ajouter une adresse ?

Trois façons : (1) Collez un lien Apple Maps — copiez un lien depuis Apple Maps et collez-le dans Navve ; le nom, l'adresse et les coordonnées sont extraits automatiquement. (2) Collez n'importe quelle adresse — collez une adresse postale complète et Navve la géocode pour vous. (3) Ajoutez manuellement — remplissez vous-même les détails dans le formulaire.

Où mes données sont-elles stockées ?

Vos données sont stockées dans votre compte iCloud personnel (si iCloud est activé pour Navve) ou uniquement sur votre appareil (si iCloud est désactivé). Nous n'avons jamais accès à vos adresses. Consultez notre Politique de confidentialité pour plus de détails.

Navve se synchronise-t-il entre les appareils ?

Navve se synchronise sur votre iPhone et vos autres appareils iOS via iCloud. Assurez-vous qu'iCloud est activé pour Navve dans Réglages → Identifiant Apple → iCloud → Navve. Les modifications se synchronisent automatiquement en arrière-plan — aucune action manuelle nécessaire. Une application compagnon pour macOS arrive au T2 2026.

Quelle est la différence entre Gratuit et Pro ?

Les deux formules donnent accès à toutes les fonctionnalités — collections, recherche, filtres, itinéraires, synchronisation iCloud, et plus. La seule différence est la limite d'adresses : Gratuit permet d'enregistrer jusqu'à 8 adresses, Pro offre un nombre illimité.

Comment fonctionne l'achat Pro ?

Pro est un achat unique de 4,99 $ — pas d'abonnement, pas de frais récurrents. Payez une fois et c'est à vous pour toujours, y compris sur tous vos futurs appareils connectés au même identifiant Apple. Utilisez Réglages → Restaurer l'achat si vous réinstallez l'app ou changez d'appareil.

Comment exporter mes adresses ?

Ouvrez Navve → Réglages → Exporter les adresses. Vous pouvez copier la liste formatée dans votre presse-papiers ou la partager via Messages, E-mail, AirDrop ou toute autre app. Les exports incluent le nom, la catégorie, l'adresse complète, les coordonnées, le site web, les notes et le statut.

Que se passe-t-il si je supprime l'app ?

Si vous utilisez iCloud : vos données restent en sécurité dans votre compte iCloud. Réinstallez Navve et tout revient automatiquement.
Si vous n'utilisez pas iCloud : toutes les données sont stockées uniquement sur votre appareil — supprimer l'app les supprime définitivement. Exportez d'abord vos adresses si vous souhaitez en garder une copie.

Navve suit-il ma position ?

Non. L'accès à la localisation n'est utilisé que localement sur l'appareil lorsque vous ouvrez la vue carte ou utilisez « Trier par distance ». Nous ne recevons, ne stockons ni ne suivons jamais votre position. Vous pouvez refuser l'autorisation de localisation : Navve continue de fonctionner normalement — vous n'aurez simplement plus le tri par distance.

Existe-t-il une app pour Mac ?

Une application compagnon native pour macOS arrive au deuxième trimestre 2026. Elle partagera les mêmes données iCloud que votre iPhone — ajoutez et organisez les adresses sur votre Mac, puis naviguez depuis votre téléphone.
